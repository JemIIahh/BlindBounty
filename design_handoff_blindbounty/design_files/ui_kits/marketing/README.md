# Marketing UI Kit

Recreation of the `frontend/` marketing surface. Dark + white + Fraunces serif.

- `primitives.jsx` — `Icon`, `LogoMark`, `Floret`, `Button`, `Badge`, `TypeTag`, `GlassCard`, `Navbar`, `Footer`
- `sections.jsx` — `Hero`, `SectionHeader`, `HowItWorks`, `TrustSection`, `RoleChooser`
- `index.html` — assembled landing page

All components are exposed to `window` at end of each file so Babel scripts can share them.
